second section

This bot is for manage server discord
The main feature is associate discords players with steam accounts in order to identify players in discord server.
You can use /steamconnect to request the bot to add a button to login with steam
And then the bot can assign a discord custom range.
You can automatically rename players in discord in order to have the same nicks in rust game and your discord server
You can get steamid from discord player using the command /steamid
You can save commands and apply when you wipe your server using /applycmd /savecmd
